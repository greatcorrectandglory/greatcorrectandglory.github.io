# Screenshot Replacement Guide

This directory is reserved for desensitized product screenshots.

Before publishing any screenshot, verify that it does not expose:

- customer names or product model numbers
- internal company names, factory locations, account names, or operator names
- real device addresses, gateway URLs, protocol fields, keys, tokens, or secrets
- private repository paths, branch names, local usernames, or internal file servers
- real serial numbers, UID values, MES order numbers, or production batch IDs

Recommended public screenshots:

- main test execution screen with anonymized product and station names
- product configuration screen with synthetic resource names and placeholder addresses
- result or log screen with synthetic UID and generic failure text

Use PNG or WebP images and reference them from the homepage screenshot section only after review.
